# Final Report Template

## Findings

List validated findings first, sorted by severity.

For each finding:
- `[severity] title`
- `file:line`
- `why this is a bug`
- `impact`
- `suggested fix direction`
- `missing/needed test`

## Open Questions / Assumptions

State unresolved assumptions that affect confidence.

## Change Summary (brief)

Give a short neutral summary of what changed.

## Residual Risk

If findings are zero, still list plausible residual risks and untested areas.
