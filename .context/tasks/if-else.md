# if/else Statement

## Overview
Implement `if`/`else` statements.

```lak
if condition {
    // ...
} else if condition2 {
    // ...
} else {
    // ...
}
```

### Features
- Basic `if condition { ... }`
- `else` branch
- `else if` chain

### Condition
- Conditional expressions must be of type `bool`.
- Use results of comparison and logical operators.
